Introduction to ALL IN ONE Splitter

Back to top

 ALL IN ONE Splitter is a Java based tool for making our distribution of files easier in removable media like floppies, ZIP drives etc., For this it can split a large file that cannot fit into a single floppy or other media to one or more files that is easily or perfectly fit into them. Also it is possible to merge them back in destination machine.

Some of its advantages:

  1. It also have a batch file generator, which can be used to merge the splitted files back even when the ALL IN ONE Splitter is not available on the destination machine.
  2. It have a facility to include comment messages in our split files.
  3. It have the flexibility to choose our own split sizes in addtion to the predefined sizes and also choose number of splits irrespective of the size.

How to Split Files?

Back to top

For splitting one or more files, do the following:

  • First, select those files you want to split by clicking the Add button. You can also edit this list by using Remove or Remove All buttons.
  • Naturally, all files are applied the split configurations in default(120 KB each). Suppose you want to specify separate settings for each file, you have to double click on the file in the list, Now Set button appears on the right. Now you can select any predefined size or customized size or number of splits and click Set button to save it. Follow the same for other files also, if necessary.
  • Next choose the destination path where your split files needs to be created, either type the path directly or click Browse to select it
  • Select the three options on the right, if necessary
  • Type any comments you want on the textarea in the left, However it is applied only selecting one file to split.
  • Now click the Split button to make the split.

How to Merge Files?

Back to top

For merging the split files, do the following:

  • Select the ALL IN ONE Splitter Info file (*.jfs) by clicking Browse button at the top.
  • Select the destination path where the splitted files needs to be merged.
  • Click View Details to view the splitted file details.
  • Click Get Comments button to view the comments if any.
  • Click Merge button to do the Merge.It is important to have all the split files in the same folder as that of the *.jfs file.
  • Check Delete Split files after Merging if you want to do so before click the Merge button.

Configuring ALL IN ONE Splitter

Back to top

We can save the settings like default Split folder, default Merge folder and default Split configurations by using ALL IN ONE Splitter configurations under Options tab.

To set defalut folders for Split and Merge files, click the Browse button right to the textbox and choose the folder you want to set as default.ALL IN ONE Splitter will add a subfolder SplitFiles or MergedFiles to your paths. You can choose different Split and Merge folders.

Also you can choose the type of split, either predefined or your own, as well as the size or number of splits you want by default. For this you have to click Browse button right to the Default size box and select what you want. After doing the above said changes, click Save to save it for future use.
